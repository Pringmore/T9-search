
### T9-Search experiment     
In this coding project you will find a simulation for a T9 search field based on LEMP x OOPHP 
### To start Application for 1st time: 
``npm run init``
### To start Application: 
``npm run start``
### To stop Application:
``npm run down``
### To start Tests:
``npm run test``
